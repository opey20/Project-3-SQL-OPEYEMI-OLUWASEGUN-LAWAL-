# Project 3: SQL Data Analysis

## Overview

In this project, I used SQL Server Management Studio (SSMS) to analyse a cleaned sales dataset using SQL queries.

The dataset was first cleaned in Excel and Power Query before being imported into SQL Server for analysis.

The aim of the project was to practise basic SQL commands such as SELECT, WHERE, GROUP BY, ORDER BY, COUNT, SUM, and AVG while analysing sales transaction data.

---

- SQL Server Management Studio (SSMS)

- SQL Queries
- CSV Dataset Import

## SQL Analysis Performed

The following SQL operations were carried out during the project:

- Displayed all records using SELECT
- Selected specific columns from the table
- Sorted records using ORDER BY
- Filtered records using WHERE
- Grouped records using GROUP BY
- Counted records using COUNT
- Calculated total revenue using SUM
- Calculated average values using AVG
- Analysed products, coupons, and payment methods

---

## Key Findings

- Printers recorded the highest number of purchases
- Chairs generated the highest total revenue
- Laptops had the highest average order value
- Freeship was the most frequently used coupon
- Freeship generated the highest coupon revenue
- Credit Card generated the highest payment revenue
