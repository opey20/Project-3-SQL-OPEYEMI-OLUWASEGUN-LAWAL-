## SQL Queries Used

### View All Records

```sql
SELECT *
FROM Sales_Data;
```

Displays all records in the Sales_Data table.

### View Selected Columns

```sql
SELECT Product, TotalPrice
FROM Sales_Data;
```

Displays only selected columns from the table.

### Sort Records by Revenue

```sql
SELECT Product, TotalPrice
FROM Sales_Data
ORDER BY TotalPrice DESC;
```

Sorts records from highest revenue to lowest revenue.

### Product Purchase Frequency

```sql
SELECT
    Product,
    COUNT(OrderID) AS Number_of_Orders
FROM Sales_Data
GROUP BY Product
ORDER BY Number_of_Orders DESC;
```

Counts the number of orders for each product.

### Product Revenue Analysis

```sql
SELECT
    Product,
    SUM(TotalPrice) AS Total_Revenue
FROM Sales_Data
GROUP BY Product
ORDER BY Total_Revenue DESC;
```

Calculates total revenue generated by each product.

### Average Order Value by Product

```sql
SELECT
    Product,
    AVG(TotalPrice) AS Average_Order_Value
FROM Sales_Data
GROUP BY Product
ORDER BY Average_Order_Value DESC;
```

Calculates average order value for each product.

### Coupon Usage Analysis

```sql
SELECT
    CouponCode,
    COUNT(OrderID) AS Total_Orders
FROM Sales_Data
GROUP BY CouponCode
ORDER BY Total_Orders DESC;
```

Counts how frequently each coupon code was used.

### Coupon Revenue Analysis

```sql
SELECT
    CouponCode,
    SUM(TotalPrice) AS Coupon_Revenue
FROM Sales_Data
GROUP BY CouponCode
ORDER BY Coupon_Revenue DESC;
```

Calculates revenue generated by each coupon code.

### Payment Method Revenue Analysis

```sql
SELECT
    PaymentMethod,
    SUM(TotalPrice) AS Payment_Revenue
FROM Sales_Data
GROUP BY PaymentMethod
ORDER BY Payment_Revenue DESC;
```

Calculates revenue generated by each payment method.

### Delivered Orders by Payment Method

```sql
SELECT
    PaymentMethod,
    COUNT(OrderID) AS Delivered_Orders
FROM Sales_Data
WHERE OrderStatus = 'Delivered'
GROUP BY PaymentMethod
ORDER BY Delivered_Orders DESC;
```

Shows the number of delivered orders by payment method.

## Key Findings

The analysis revealed several useful business insights:

* Printers recorded the highest number of purchases.
* Chairs generated the highest total revenue.
* Laptops had the highest average order value.
* Freeship was the most frequently used coupon code.
* Freeship generated the highest coupon-related revenue.
* Credit Card generated the highest payment method revenue.
* Product performance varied significantly across categories.
* Customer purchasing behaviour differed depending on coupon and payment method usage.

---

## Recommendations

Based on the analysis, the following recommendations can be made:

* Focus marketing efforts on high-performing products such as Chairs and Printers.
* Promote successful coupon campaigns such as Freeship to encourage customer purchases.
* Investigate opportunities to increase sales of lower-performing products.
* Encourage the use of high-value payment methods through targeted promotions.
* Continue monitoring product performance and customer purchasing patterns using SQL-based reporting.
