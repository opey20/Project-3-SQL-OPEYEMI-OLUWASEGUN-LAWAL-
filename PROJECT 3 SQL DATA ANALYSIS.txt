PROJECT 3: SQL DATA ANALYSIS

Project Overview
This project focused on performing SQL-based data analysis using Microsoft SQL Server Management Studio (SSMS). The cleaned transactional dataset generated from previous projects was imported into a relational database table and analyzed using SQL queries.

The aim of the project was to use SQL querying techniques to extract business insights, summarize transactional data, and demonstrate foundational database analysis skills.

SQL QUERIES PERFORMED

1. Display All Records

Query:
SELECT *
FROM Sales_Data;

Purpose:
This query was used to display all records and columns from the Sales_Data table.

--------------------------------------------------

2. Display Specific Columns

Query:
SELECT Product, TotalPrice
FROM Sales_Data;

Purpose:
This query displayed only the Product and TotalPrice columns from the dataset.

--------------------------------------------------

3. Sort Highest Revenue Transactions

Query:
SELECT Product, TotalPrice
FROM Sales_Data
ORDER BY TotalPrice DESC;

Purpose:
This query sorted transactions from the highest revenue to the lowest revenue.

--------------------------------------------------

4. Product Purchase Frequency Analysis

Query:
SELECT Product,
COUNT(OrderID) AS Number_of_Orders
FROM Sales_Data
GROUP BY Product
ORDER BY Number_of_Orders DESC;

Purpose:
This query counted the number of orders for each product and sorted the products based on purchase frequency.

Insight:
Printers recorded the highest purchase frequency among all products.

--------------------------------------------------

5. Product Revenue Analysis

Query:
SELECT Product,
SUM(TotalPrice) AS Total_Revenue
FROM Sales_Data
GROUP BY Product
ORDER BY Total_Revenue DESC;

Purpose:
This query calculated the total revenue generated by each product.

Insight:
Chairs generated the highest overall revenue within the dataset.

--------------------------------------------------

6. Average Product Order Value

Query:
SELECT Product,
AVG(TotalPrice) AS Average_Order_Value
FROM Sales_Data
GROUP BY Product
ORDER BY Average_Order_Value DESC;

Purpose:
This query calculated the average order value for each product category.

Insight:
Laptop purchases recorded the highest average transaction value.

--------------------------------------------------

7. Coupon Usage Analysis

Query:
SELECT CouponCode,
COUNT(OrderID) AS Total_Orders
FROM Sales_Data
GROUP BY CouponCode
ORDER BY Total_Orders DESC;

Purpose:
This query analyzed the frequency of coupon usage across customer orders.

Insight:
The Freeship coupon was the most frequently used promotional offer.

--------------------------------------------------

8. Coupon Revenue Analysis

Query:
SELECT CouponCode,
SUM(TotalPrice) AS Coupon_Revenue
FROM Sales_Data
GROUP BY CouponCode
ORDER BY Coupon_Revenue DESC;

Purpose:
This query calculated the total revenue generated by each coupon campaign.

Insight:
Orders using the Freeship coupon generated the highest revenue.

--------------------------------------------------

9. Payment Method Revenue Analysis

Query:
SELECT PaymentMethod,
SUM(TotalPrice) AS Payment_Revenue
FROM Sales_Data
GROUP BY PaymentMethod
ORDER BY Payment_Revenue DESC;

Purpose:
This query evaluated the revenue performance of different payment methods.

Insight:
Credit Card transactions generated the highest total revenue.

--------------------------------------------------

10. Delivered Orders Analysis

Query:
SELECT PaymentMethod,
COUNT(OrderID) AS Delivered_Orders
FROM Sales_Data
WHERE OrderStatus = 'Delivered'
GROUP BY PaymentMethod
ORDER BY Delivered_Orders DESC;

Purpose:
This query filtered only delivered orders and analyzed the payment methods associated with successful deliveries.
